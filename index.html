MunchyFuel

MunchyFuel is a Rails application that randomly generates a picture from a restaurant and delivery information. It is integrated with Foursquare's API to randomly select restaurants nearby based on the user's location. Users can decide to "Want" or "Don't Want" the food, or flag the picture as irrelevant. Reported irrelevant photos will not be displayed again on MunchyFuel.

View live site Github
  • Technologies: Ruby on Rails, JavaScript, jQuery, Foursquare API, Geocoder, Bootstrap

Scoutr

Scoutr is a Rails application integrated with the Yelp API that creates custom itineraries with activities for the day based on user inputs and location. Users will have the ability to sort, replace, or delete activities recommended by Scoutr. Scoutr can also search for itineraries generated by other users.

View demo Github
  • Technologies: Ruby on Rails, JavaScript, AJAX, jQuery, Yelp API, Google Maps API, Geocoder, Fuzzy Search, Bootstrap

QraveIt

QraveIt is a Rails application that allows users to search and write reviews for specific dishes at restaurants. The rating system is simple and uses a thumbs up, unsure, and thumbs down rating for a dish and reviews are limited to a certain number of characters to be short and sweet. The suggestive search makes it easy for users to easily find a dish they are craving, or quickly view the top rated dishes at a restaurant.

View live site Github
  • Technologies: Ruby on Rails, JavaScript, AJAX, jQuery, Elastic Search, Typeahead.js, Geocoder, Google Maps API, Google Places API, RSpec, Bootstrap
